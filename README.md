# Nekko Manga Translator — интерактивный инструмент для подготовки перевода манги

Инструмент, который сильно ускорит и упроcтит подготовку перевода  вашей манги. Эта программа не даёт готовый результат, вы сохраняете полный контроль над каждым этапом и качеством финального продукта. Программа берёт на себя самую тяжёлую и нудную часть работы: обнаружение и перенос текста с оригинала. Дальше либо вы вручную переводите текст в удобной программе, либо доверяете это продвинотому ИИ, который специально настроен на понимание контекста и персонажей, а не простой роботизированный перевод.

Полностью автоматизировать верстку манги так, чтобы она всегда выглядела идеально, пока невозможно — сделать сделать идеал, это всё ещё ручная работа.

Nekko Manga Translator позволяет сократить часы монотонной подготовки до минут и сосредоточиться на том, что действительно важно — на качестве перевода и оформлении. В результате вы получите файл, который удобно открыть в Adobe Photoshop и быстро перенести текст. Программа ничего не «закрашивает» и не портит изображение — она работает только с разметкой и текстом.

## Как работает

1. Загружаете страницы манги.
2. Детектор автоматически находит все пузыри и области с текстом
3. Вы подтверждаете или редактируете эти области.
4. Запускаете распознавание текста.
5. Полученный текст отображается в удобном редакторе,.
6. Дальше вы можете поправить или отправить на перевод/редактуру другому пользователю. Либо воспользоваться продвинутым ИИ-переводом.

## Модели

* comic-text-and-bubble-detector — детектор пузырей / текста (RT-DETR).
* manga-ocr через comic-translate — OCR для японского.
* PaddleOCR-VL-For-Manga — OCR для разных языков (требуется видеокарта с 8+ гигабайтами видеопамяти).

## Предупреждение

* Проект сырой и написан начинающим программистом.
* Интерфейс и API могут меняться.

## Установка

Установите Python 3.12.х

### Windows

Запустите `webui.bat` из папки проекта (НЕ от имени Аддминистратора).

Может потребоваться запутить файл несколько раз.

Если страница не открылась, перейдите на `http://localhost:5173`.

### Linux / macOS

Запустите в терминале в папке проекта:

```bash
./webui.sh
```

Может потребоваться запутить файл несколько раз.

Если страница не открылась, перейдите на `http://localhost:5173`.

Примечания:

* Не запускайте `webui.bat` от имени администратора.
* На первом запуске возможен запрос прав администратора для установки Git / Node.
* В России для скачивания МОЖЕТ понадобиться VPN (но рекомендуется избегать его применения).
